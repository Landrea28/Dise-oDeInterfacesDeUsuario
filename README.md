## Proyecto: Melodia Stream - Plataforma de Streaming Especializada

Este README corresponde al **Taller 2** de la materia Diseno de Interfaces de Usuario.
El proyecto se encuentra en la rama **MelodiaSteam** dentro de un repositorio en GitHub.

## Como esta organizado el repositorio

La idea del repositorio es mantener cada avance academico de forma ordenada por ramas, para que cualquier persona externa pueda revisar facilmente cada entrega.

- Cada taller/laboratorio se gestiona en su propia rama.
- Esta rama, **MelodiaSteam**, contiene especificamente el contenido del **Taller 2**.
- En cada rama existe un README que explica que trabajo contiene, su contexto y sus archivos principales.

## Objetivo de este taller

Disenar y desarrollar una interfaz web moderna para una plataforma de streaming de musica enfocada en tres generos: **Alternativa, Jazz y Rock**.
El enfoque del trabajo es aplicar principios de:

- Usabilidad.
- Accesibilidad (WCAG AA).
- Diseno responsivo.
- Consistencia visual bajo una estetica retro-vintage.

## Estructura de archivos de esta rama

- **index.html:** Portal principal con exploracion de generos y artistas recomendados.
- **quienes-somos.html:** Narrativa de marca sobre la curaduria humana y la pasion por el sonido sin perdida.
- **productos-servicios.html:** Planes de suscripcion y servicios exclusivos como "Vinyl Club".
- **login.html:** Interfaz de acceso de usuario con diseno minimalista.
- **styles.css:** Estilos globales del proyecto (paleta, tipografia, componentes y responsividad).

## Tecnologias usadas

- HTML5 semantico.
- CSS3 puro (Flexbox, Grid, Variables, Media Queries).

## Criterios UX/UI aplicados

1. **Accesibilidad:** Contrastes de color y navegacion optimizada para teclado.
2. **Responsividad:** Adaptacion a movil, tablet y escritorio.
3. **Consistencia visual:** Uso de variables CSS para identidad de marca.
4. **Interactividad:** Feedback visual en botones, formularios y tarjetas.
